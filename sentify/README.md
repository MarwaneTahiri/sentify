# sentify

A web application that analyses the emotional tone 
of any text. Paste in a sentence, paragraph, or 
article and Sentify will tell you whether it's 
positive, negative, or neutral — with a confidence 
score and a breakdown of the key phrases driving 
the result.

Built with Python, Flask, and a HuggingFace 
pre-trained transformer model.

Stack: Python · Flask · HuggingFace Transformers · 
SQLite · HTML/CSS/JS
